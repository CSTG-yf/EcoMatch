# BE-10 业务化解释与图表推荐服务实现说明

## 已实现

- 新增确定性业务分析处理器，随 Chat 查询结果返回主推荐图、候选图、推荐理由、置信度和字段映射。
- 提供独立的图表推荐接口 `POST /api/chat/insight/recommend` 和业务解释接口 `POST /api/chat/insight/explain`，接口访问前统一校验登录态。
- 支持 `KPI_CARD`、`LINE`、`BAR`、`PIE`、`COMBO`、`TABLE` 六类图表。
- 图表规则约束饼图分类数、非负值、多指标组合图和多维明细表，避免误导性可视化。
- 对“占比、构成、组成、份额、比例”等问题识别构成分析意图；仅在单分类、单指标、非负值且分类数受限时推荐饼图。
- 业务解释仅从真实结果生成数值范围、首末变化、时间范围、统计异常候选和分类贡献度。
- 趋势及组合查询显式引用首条和末条记录值，避免只给变化率而缺少原始证据。
- 接入指标名称、描述和单位作为口径说明，并保留原始问题作为分析范围。
- 少于 3 条、空值、脱敏值和不可解析值不会输出确定性趋势结论。
- 空结果返回低置信度表格建议和“无数据、不输出结论”说明，不静默缺失或抛出业务异常。
- 小样本只保留原值、范围和可复算的所选范围占比，不输出首末变化或异常判断。
- 独立接口校验声明字段与结果字段一致性，并能在首行为空时从后续记录识别数值字段。
- 对无分类维度的月度序列，从真实月份和值计算最新月份的环比及同比；基期缺失或为零时不输出对应结论。
- 所有解释附带范围外推限制，风险指标附加监管、授信和风险处置免责声明。
- 风险类指标自动附加监管报送、授信审批和风险处置免责声明。
- 确定性解释先于可选大模型解释执行，确保关键数值和结论可校验。
- 结果返回前执行独立一致性校验：图表字段必须来自查询列，置信度必须处于 `[0,1]`，时间范围、记录数、证据、警告和口径必须与摘要一致。
- 一致性校验同时拒绝主推荐图或任一候选图引用脱敏字段，并在证据复算和时间范围校验中排除脱敏列。
- 独立复算数值范围、首末记录、最新值、异常候选、变化率和贡献度；无法从查询结果复算的证据直接终止输出，避免错误结论进入前端或大模型链路。
- 独立接口限制输入结果的最大行数和列数，超限请求在进入解释处理前直接拒绝，避免超大结果占用服务内存和计算资源。
- 数值解析限制最大精度和绝对 scale，NaN、Infinity 及可能展开为超长字符串的极端科学计数值按不可解析处理，不进入范围、变化率、异常点或贡献度计算。
- 图表推荐仅使用可解析且未脱敏的数值字段；全部数值不可用时降级为低置信度表格，不再生成误导性的数值图候选。
- 缓存命中与非缓存查询使用一致的脱敏状态和字段元数据；缓存响应采用独立快照，解释或图表处理不会修改后续请求的数据依据。

## 请求字段

- `queryText`：原始业务问题。
- `queryColumns`：查询结果字段及类型。
- `queryResults`：已完成权限过滤和脱敏的查询结果。
- `metrics`：命中的指标定义、描述和单位。

## 配置

| 配置项 | 默认值 | 说明 |
| --- | --- | --- |
| `s2.business-insight.small-sample-threshold` | `3` | 小样本警告阈值 |
| `s2.business-insight.pie-max-categories` | `6` | 饼图最大分类数 |
| `s2.business-insight.anomaly-z-score` | `2.0` | 异常点 Z-Score 阈值 |
| `s2.business-insight.low-confidence` | `0.65` | 低置信度 |
| `s2.business-insight.evidence-confidence` | `0.82` | 存在可校验证据时的置信度 |
| `s2.business-insight.high-confidence` | `0.95` | 高质量结果置信度 |
| `s2.business-insight.max-input-rows` | `10000` | 独立接口允许的最大结果行数 |
| `s2.business-insight.max-input-columns` | `100` | 独立接口允许的最大结果列数 |

## 响应字段

- `recommendedChart`：主推荐图。
- `candidateCharts`：候选图列表。
- `businessExplanation`：摘要、证据、指标口径、时间范围、警告和置信度。

## 验证

- `BusinessInsightProcessorTest` 覆盖六类图表、构成问题意图、负值饼图保护、脱敏字段排除、不可解析数值图表降级、极端数值资源保护、真实数值引用、可复算同比环比、小样本、空结果、首行空值、贡献度和风险提示。
- `BusinessInsightConsistencyValidatorTest` 覆盖未知图表字段、主图或候选图引用脱敏字段、伪造数值证据和错误时间范围的 fail-closed 拒绝。
- `BusinessInsightServiceTest` 覆盖独立推荐/解释接口服务、可配置分类阈值、字段契约校验、空结果说明及输入行列上限。
- 测试直接读取 DATA-03 冻结测试集，图表匹配准确率门禁不低于 90%。
- 冻结测试集自动验证必需数值、时间范围、问题范围和风险声明，解释覆盖率门禁不低于 90%。
- DATA-03 校验器通过 90 条样本、30 条问卷、来源哈希和数据隔离检查。
