# DATA-03 图表与业务解释数据集实现说明

## 交付范围

- 基于比赛 Excel 的真实指标值生成 90 条图表与解释样本，训练、验证和测试集各 30 条。
- 指标卡、折线、柱状、饼图、组合图和表格各 15 条，覆盖经营分析、风险管控和客户营销场景。
- 每条样本标注数据结构、推荐图表、候选图表、不推荐原因、推荐理由和置信度。
- 每条解释绑定真实结果数值、单位、指标定义版本、时间范围、机构范围、风险提示和禁止结论。
- 提供 30 条业务理解度问卷，采用 10 分制评分，9 分为通过线，对应不低于 90% 的验收要求。

## 安全边界

- 饼图只用于同单位、正值且不超过 6 项的所选范围构成，禁止外推为全量业务占比。
- 少于 3 个观测值时仅允许陈述事实，不输出趋势、异常点或因果结论。
- 风险指标必须声明不替代监管报送、授信审批或风险处置。
- 解释中的关键数值必须存在于查询结果，且必须同时出现单位、指标口径和时间范围。

## 自动验收

```bash
python evaluation/bank_chart_explanation/build_dataset.py
python evaluation/bank_chart_explanation/validate_dataset.py
```

详细字段、生成物和评测说明见 `evaluation/bank_chart_explanation/README.md`。
